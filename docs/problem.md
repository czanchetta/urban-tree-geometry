# Problema

Projetos luminotécnicos externos precisam representar a vegetação para avaliar
obstrução, sombreamento e aparência. No DIALux e ferramentas similares, uma
árvore é um objeto 3D com altura e dimensões de copa.

Na fase de projeto, porém, raramente há levantamento dendrométrico de campo.
Com frequência o único dado disponível é a **espécie** e o **perímetro do
tronco** registrados em um inventário.

Este projeto responde à pergunta: *como obter parâmetros geométricos razoáveis e
tecnicamente transparentes para pré-modelagem, a partir apenas desse mínimo, sem
apresentar o resultado como uma medição de campo?*
