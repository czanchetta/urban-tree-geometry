# Problem

Outdoor lighting projects must represent vegetation to assess obstruction,
shading and appearance. In DIALux and similar tools a tree is a 3D object with a
height and crown dimensions.

At the design stage, however, a field dendrometric survey is rarely available.
Often the only data are the **species** and the **trunk perimeter** recorded in
an inventory.

This project answers: *how to obtain reasonable, technically transparent
geometry parameters for pre-modelling from that minimum alone, without presenting
the result as a field measurement?*
